Flex Container Properties :-

display: flex
This will enable to apply all CSS flexbox properties and by default it align all the items in a row order.
flex-direction: row(default)
This property allow to align the items either in a row or column. It's default value on the web i.e main-axis is row.
flex-wrap: nowrap(default)
The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.
flex-flow - row nowrap(default)
The flex-flow CSS property is a shorthand property of flex-direction and flex-wrap.
justify-content: flex-start(default)
The CSS justify-content property defines how the browser distributes space between and around content items along the main-axis of a flex container. Note :- The property work and align the item on the main axis.ie. left to right.
align-items: stretch(default)
The CSS align-items property sets the align-self value on all direct children as a group. In Flexbox, it controls the alignment of items on the Cross Axis i.e left to right.
align-content: stretch(default)
The CSS align-content property sets the distribution of space between and around content items along a flexbox's cross-axis. Note :- But align-content is for multi line flexible boxes. It has no effect when items are in a single line. It aligns the whole structure according to its value.

Flex Items Properties:-

order:-
By default, flex items are laid out in the source order. However, the order property controls the order in which they appear in the flex container.
flex-grow :-0(default)
This defines the ability for a flex item to grow if necessary. It accepts a unitless value that serves as a proportion.
flex-shrink:-1(default)
This defines the ability for a flex item to shrink if necessary.
flex-basis:-auto(default)
This defines the default size of an element before the remaining space is distributed. It can be a length (e.g. 20%, 5rem, etc.) or a keyword.
align-self:-stretch(default)
This defines the default size of an element before the remaining space is distributed. It can be a length (e.g. 20%, 5rem, etc.) or a keyword.
